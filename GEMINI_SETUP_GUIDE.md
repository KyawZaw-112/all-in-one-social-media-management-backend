# Google Gemini Free API Setup Guide

To use the AI features of Vibe for free, follow these steps to get your **Gemini API Key**:

### 1. Go to Google AI Studio
Visit [https://aistudio.google.com/](https://aistudio.google.com/) and sign in with your Google account.

### 2. Create an API Key
- On the left sidebar, click on **"Get API key"**.
- Click the button **"Create API key in new project"**.
- Wait for it to generate, then **Copy** the API key.

### 3. Add to your Project
Open your `.env` file in the `backend` folder and add the following line at the bottom:

```env
GEMINI_API_KEY=ယခုလေးတင်ကော်ပီကူးလာသော_key_ကို_ဒီမှာထည့်ပါ
```

### 4. Provide the Key to me
Once you have added it to the `.env` file, just let me know "Key ထည့်ပြီးပြီ" (Key added) and I will proceed with the implementation.
